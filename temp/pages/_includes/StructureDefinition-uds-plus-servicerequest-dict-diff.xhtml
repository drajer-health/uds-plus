1. ServiceRequest
Definition

The UDS+ profile is based on the ServiceRequest resource and establishes the core elements, extensions, vocabularies and value sets for representing ServiceRequests for UDS+ program.

A record of a request for service such as diagnostic investigations, treatments, or operations to be performed.

2. ServiceRequest.id
Definition

De-dentified ServiceRequest resource identifier.

The logical id of the resource, as used in the URL for the resource. Once assigned, this value never changes.
