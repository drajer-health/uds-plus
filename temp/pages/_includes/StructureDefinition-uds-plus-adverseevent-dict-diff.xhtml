1. AdverseEvent
Definition

The UDS+ profile is based on the AdverseEvent resource and establishes the core elements, extensions, vocabularies and value sets for representing AdverseEvent for UDS+ program.

Actual or potential/avoided event causing unintended physical injury resulting from or contributed to by medical care, a research study or other healthcare setting factors that requires additional monitoring, treatment, or hospitalization, or that results in death.

2. AdverseEvent.id
Definition

De-dentified AdverseEvent resource identifier.

The logical id of the resource, as used in the URL for the resource. Once assigned, this value never changes.
