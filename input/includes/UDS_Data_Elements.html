UDS Table : Patients By Zip Code Table
Zip Code uds-plus-patient.address.postalCode None
Insurance Type uds-plus-coverage.type Value Set specific to UDS+ reporting
UDS Table 3A: Patients By Age and By Sex Assigned at Birth
Age uds-plus-patient.extension (uds-plus-plus-age-extension) Populated using the birthDate of the patient.
Sex Assigned at Birth uds-plus-patient.extension (us-core-birthsex) NullFlavor values are not allowed for UDS+ reporting.
UDS Table 3B: Demographic Characteristics
Race uds-plus-patient.extension (us-core-race) Require both ombCategory and Detailed Race codes along with Null Flavors
Ethnicity uds-plus-patient.extension (us-core-ethnicity) Require both ombCategory and Detailed Ethnicity codes along with Null Flavors
Preferred Language uds-plus-patient.communication.language None
Sexual Orientation uds-plus-sexual-orientation-observation.valueCodeableConcept Additional NullFlavor concept of NASK has to be supported
Gender Identity uds-plus-patient.extension (us-core-genderIdentity) Additional NullFlavor concept of NASK has to be supported
