UDS Data Element UDS+ Reporting FHIR Profile Mapping Differences with US Core - 6.1.0/USCDIv3
UDS Table : Patients By Zip Code Table
Zip Code de-identified-uds-plus-patient.address.postalCode None
Insurance Type uds-plus-coverage.type Value Set specific to UDS+ reporting
UDS Table 3A: Patients By Age and By Sex Assigned at Birth
Age de-identified-uds-plus-patient.extension (uds-plus-plus-age-extension) Populated using the birthDate of the patient.
Sex Assigned at Birth de-identified-uds-plus-patient.extension (uds-plus-birthsex-extension) NullFlavor values are not allowed for UDS+ reporting.
UDS Table 3B: Demographic Characteristics
Race de-identified-uds-plus-patient.extension (uds-plus-race-extension) Require both ombCategory and Detailed Race codes for UDS+ reporting
Ethnicity de-identified-uds-plus-patient.extension (uds-plus-ethnicity) Require both ombCategory and Detailed Ethnicity codes for UDS+ reporting
Preferred Language de-identified-uds-plus-patient.communication.language None
Sexual Orientation uds-plus-sexual-orientation-observation.valueCodeableConcept Additional NullFlavor concept of NASK has been added for UDS+ reporting
Gender Identity uds-plus-patient.extension (uds-plus-gender-identity-extension) Additional NullFlavor concept of NASK has to be supported
UDS Table 4: Selected Patient Characteristics
Income uds-plus-income-observation.code & uds-plus-income-observation.valueRange Income is modeled as an observation and is not present in US Core.
Insurance uds-plus-coverage.type Value Set specific to UDS+ reporting
Managed Care Utilization uds-plus-coverage.extension (uds-plus-capitated-member-months-extension, uds-plus-feeforservice-member-months-extension) Extensions added for UDS+ reporting as they are not present in US Core
Special Population - Migratory/Seasonal worker uds-plus-patient.extension (uds-plus-plus-agriculture-worker-status) Extension added for UDS+ reporting as it is not present in US Core
Special Population - Homeless uds-plus-patient.extension (uds-plus-plus-housing-status) Extension added for UDS+ reporting as it is not present in US Core
Special Population - Veterans uds-plus-patient.extension (uds-plus-plus-veteran-status) Extension added for UDS+ reporting as it is not present in US Core
